### 기능 단순 정리

---

**여기서부터 기능 적어주시면 됩니다**

- 시간표
  - 커리큘럼 확인
- 프로젝트 팀원 or (교육생) 위치 공유
- 출결확인
  - 입실 및 퇴실 알람 기능
- 점심 메뉴 확인
  - 오늘의 메뉴 후기 남기기
- 프로님과 화상 삼당(Web RTC)
  - 담당 프로님과 교육생 모두 승인 시 진행
- 설문조사 등 todo 리스트 기능
- 마일리지 및 경험치 레이팅 기능(내 순위만 확인 가능)
- 팀구성
  - 팀관리
  - 팀원 현황
  - 프로젝트 관련 게시판(회의록, gantt chart, 명세서 등)
- 강의 다시보기
- 공지사항 확인
  - 새 공지사항에 대한 알림 기능
- 멘토링관련 게시판 이용
